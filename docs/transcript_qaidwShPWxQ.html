The KJV can be confusing at times


Transcript

the first decade of my preaching ministry I prepared all of my sermons in King James text simply because it was too much work if I wanted to preach this sermon somewhere that demanded King James only. There are some downsides to that because there are archaic expressions in the King James that don't communicate so you have to take care to explain those things.

Words that have changed meaning since 1611, those things all need to be explained so it was a little more work to preach actually, from the King James.
